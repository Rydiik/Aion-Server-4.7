Установка:
----------------------------------------------------
!!! Если нужно обновиться с предыдущих версий и сохранить добавленные новости и вещи из магазина - сначала открываем Переезд 2.5-3.0 !!!
Создать базу данных с названием pow (или любым другим)
Залить в созданную базу данных файл pow.sql
Залить в базу данных логин-сервера файл login.sql
Залить в базу данных гейм-сервера файл game.sql
Отредактировать файл \protected\config\config.php - указать homeUrl главной страницы, указать свои имена баз данных, логин, пароль
Залить файлы на сервер в бинарном (двоичном) режиме
Установить права на запись у папок: assets, userbars, \protected\runtime




Информация для подключения Робокассы:
----------------------------------------------------
На сайте Робокассы, во вкладке Администрирование, указываем следующие данные:
Result URL: http://ваш_сайт.ru/pay/roboresult/
Метод отсылки данных по Result URL: POST
Success URL: http://ваш_сайт.ru/pay/robosuccess/
Метод отсылки данных по Result URL: POST
Fail URL: http://ваш_сайт.ru/pay/robofail/
Метод отсылки данных по Result URL: POST
На странице Настройки платежных систем, заполняем поля Логин, Пароль для платежей №1, Пароль для платежей №2. Данные указываем те, которые прописывали на сайте Робокассы при создании магазина.


Информация для подключения Интеркассы:
----------------------------------------------------
На сайте Интеркассы, на странице Настройки, указываем следующие данные:
Success URL: http://ваш_сайт.ru/pay/iksuccess/
Метод передачи Success URL: POST
Fail URL: http://ваш_сайт.ru/pay/ikfail/
Метод передачи Fail URL: POST
Status URL: http://ваш_сайт.ru/pay/ikstatus/
Метод передачи Status URL: POST
Валюта, в которой магазин передает сумму плетежа: Российский рубль
Курс валюты: 0.03125 (или в зависимости от текущего курса доллара)
На странице Настройки платежных систем, заполняем поля Идентификатор магазина (ik_shop_id), Секретный ключ (secret_key). Данные указываем те, которые прописывали на сайте Интеркассы при создании магазина.
